Event Publishing Extensions to iCalendar


This proposal was submitted to the IETF as an Internet Draft for the Standards Track as of the date shown above, and may be found at the following URL:

https://datatracker.ietf.org/doc/draft-douglass-calendar-extension

This specification introduces a number of new iCalendar properties which are of particular use for event publishers and in social networking.
